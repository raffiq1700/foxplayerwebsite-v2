AliceBlue and Shoonya are two of the most commonly used brokers among Indian algo traders, largely because both offer free or low-cost API access — a rarity among Indian discount brokers. Here's how they compare for someone building an automated strategy.

## API access and cost

Both AliceBlue and Shoonya provide API access at no additional cost beyond standard brokerage and exchange charges. Neither charges a separate "algo trading" subscription fee at the broker level — the cost differences traders usually run into come from the tooling layer they build or buy on top (strategy builders, webhook bridges, risk management), not the broker API itself.

## Documentation and developer experience

Both brokers publish API documentation covering REST endpoints for order placement, positions, and market data.
